# BiXBiT — Digital Marketing Plan & Report

Интерактивный трекер задач и отчётов отдела digital-маркетинга BiXBiT.

## Возможности

- 📋 **План** — задачи по направлениям с редактируемыми полями
- 📊 **Отчёт** — синхронизируется из плана, фактические результаты отдельно
- 🔄 **Supabase** — все изменения сохраняются автоматически
- 🌙 **Тёмная тема** — переключатель в шапке
- ✦ **AI Отчёт** — генерация executive summary одной кнопкой
- 📋 **Боковая панель** — детали задачи по клику
- ➕ **Новый период** — создание Q3, Q4 и т.д.

## Stack

- Vanilla JS + HTML/CSS
- [Supabase](https://supabase.com) — база данных
- GitHub Pages — хостинг

## Deploy

Просто включи GitHub Pages для этого репозитория:
**Settings → Pages → Source: Deploy from branch → main → / (root)**
